# Spring 으로 만든 강의소개 사이트
 * 로그인, 회원가입 
 * 간단한 블로그 형식의 게시판 글쓰기(에디터 기능 포함), 삭제구현
 * 관리자 이메일 보내기(구글이메일 연동)
 
# 개발언어 및 환경
* Java & Spring3, Mysql
